# AI-Facial-Recognition-System-Software-Engineering-Project
Designed a full software architecture for an AI‑powered facial recognition system used to identify missing persons in real time.

Built system structure using MVC architecture, separating AI models, system logic, and user interface for scalability and maintainability.

Modeled the system using UML diagrams (use case, class, sequence, and state diagrams) to define system behavior and interactions.

Applied key design patterns including Observer for real‑time alerts and Factory for AI model integration.

Engineered a modular, high‑performance system supporting multi‑camera input, instant notifications, and secure database operations.
